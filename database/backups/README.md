# Database Backups

This folder stores SQL backups generated by `backup.sh`.

## Run backup (inside database container)

```bash
backup.sh
```

## Optional environment variables

- `BACKUP_DIR` (default: `/var/backups/postgres`)
- `DB_HOST` (default: `localhost`)
- `DB_PORT` (default: `5432`)
- `DB_USER` (default: `postgres`)
- `DB_LIST` (default: `raushni_backend raushni_cms`)

## Restore

Use `restore.sh` shipped in this folder.
